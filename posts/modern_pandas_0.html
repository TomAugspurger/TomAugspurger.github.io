Setup¶

Use conda. Either miniconda if you're a minimalist like me, or the full Anaconda distribution if you like things to be easy. conda will take care of managing environments so you can have multiple installations of python and manages all the packages within each environment. To make sure you're ready for this, just call

conda create -n py3 python=3.4 pandas ipython ipython-notebook seaborn

Pretty easy.

Data Structures¶

pandas has two main Data structures you need to worry about. DataFrame and Series.

DataFrames are for 2-dimensional, tabular data. These contain your data and define a ton of useful methods.

Series are the 1-dimensional version of DataFrames. Why the distinction? Certain methods only make sense on a 1-dimensional structure.
